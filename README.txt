Локальная копия сайта: https://www.sashakalashnikov.com/
Сохранено: март 2025

Как открыть:
  - Главная:     откройте index.html
  - Портфолио:  откройте portfolio/index.html
  - AliExpress: откройте portfolio/aliexpress/index.html

Структура:
  - index.html                    — главная страница
  - portfolio/index.html          — страница портфолио (кейсы)
  - portfolio/aliexpress/index.html — кейс AliExpress Redesign
  - assets/background.jpeg        — фоновое изображение (главная)
  - assets/favicon.png            — иконка сайта

Примечание: Сайт на Framer; шрифты и скрипты подгружаются из интернета. При офлайне контент и вёрстка отобразятся, шрифты могут быть системными. Переходы между страницами через меню могут вести на сайт — для навигации по локальной копии открывайте нужный index.html из папок выше.
